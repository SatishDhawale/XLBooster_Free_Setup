# XLBooster_Free_Setup
Download XLBooster Free Setup and Install  


> [!WARNING]  
> **Note:** This is a Microsoft Excel Add-in. Google Chrome may show a warning — this is expected. Please download using Microsoft Edge browser.


1) Download and UnZip folder

2) Once Unzip --> 
    Right --> click on the setup file
    Click Properties
    In the General tab, look at the bottom
    If you see <b> “Unblock”, </b> check the box
    Click Apply → Click OK

3) Now close Microsoft Excel before starting the installation.

4) Double-click the downloaded installer file to launch the installation wizard.

5) In the installer wizard, follow the on-screen prompts — read and accept any license agreement if shown, then click Next.

6) Choose the installation location if prompted (or use the default path), then click Install to begin.

7) Wait for the installation to complete, then click Finish.

8) Open Microsoft Excel — you should now see the XLBooster tab or add-in available in the ribbon.

9) These steps reflect the common process for installing Excel add-in tools like XLBooster and similar products.
